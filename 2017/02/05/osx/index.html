OS X一些设置

发表于 2017-02-05

修改brew源

如果出现 fatal: No such remote ‘origin’
说明没有设定remote，可以执行 git remote -v 查看，应该返回空
那么执行以下语句

设置DNS

将无线网卡使用本地DNS

  • 网络接口查看方式
    networksetup -listallnetworkservices
  • 查看当前设置的DNS
    networksetup -getdnsservers Wi-Fi
  • 清空当前DNS（自动获取）
    sudo networksetup -setdnsservers Wi-Fi empty

nodejs

  • install node and npm
    brew install node
  • update node and npm
    brew update
    brew upgrade node
  • uninstall node and npm
    brew uninstall node

npm install

npm install -g hexo-cli –https-proxy=http://127.0.0.1:4411 –proxy=http://127.0.0.1
or
npm install –registry=https://registry.npm.taobao.org
-g 全局安装
去掉-g 当前目录安装
去掉-g 和库名 安装json文档下所有库

git

git clone https://xxx –config “https.proxy=proxyHost:proxyPort”

bash自动补全

httpp://davidalger.com/development/bash-completion-on-os-x-with-brew/

读写 ntfs 方式

  • mount
    diskutil list
    mkdir ~/Desktop/APP
    sudo mount_ntfs -o rw,nobrowse /dev/disk2s5 ~/Desktop/APP
  • detach
    hdiutil detach ~/Desktop/APP
